# libGPP-Engine

libGPP-Engine es un motor gráfico experimental desarrollado en C/C++ cuyo objetivo es facilitar el desarrollo de aplicaciones gráficas y videojuegos en múltiples plataformas.

## Características

- Sistema de renderizado de sprites
- Manejo de audio
- Sistema de fuentes
- Sistema de entrada (input)
- Manejo de archivos
- Arquitectura modular

## Plataformas soportadas

El motor ha sido probado en diferentes plataformas:

- PlayStation 2
- PlayStation Portable (PSP)
- Nintendo GameCube
- PC
- Android

## Arquitectura del motor

El motor está diseñado utilizando una estructura modular que separa los diferentes componentes del sistema:

- Video / Renderizado
- Sistema de entrada
- Manejo de archivos
- Audio
- Sistema de fuentes

Esta organización facilita el mantenimiento del código y permite realizar pruebas de cada módulo de forma independiente.

## Propósito del proyecto

Este proyecto fue desarrollado con fines educativos y de investigación en el área de desarrollo de motores gráficos y portabilidad de software.
